# Smart Expense Tracker 

##  Project Overview
The **Smart Expense Tracker** is a Python-based command-line application that helps users record, manage, and analyze their daily expenses. It provides insights into spending patterns and helps users understand where their money is going.

---

##  Objectives
- Record daily expenses (date, category, amount, description)
- Categorize expenses (Food, Travel, Bills, Others)
- Generate monthly summaries
- Provide category-wise analysis
- Visualize expenses using a pie chart
- Offer basic financial insights

---

##  Technologies Used

- **Programming Language:** Python 
- **Libraries Used:**
  - `csv` → For storing and reading expense data  
  - `matplotlib` → For data visualization (pie chart)  
  - `os` → For file handling and checking file existence  

---

##  Features
-  Add Expense  
-  View All Expenses  
-  Monthly Expense Summary  
-  Category-wise Analysis  
-  Pie Chart Visualization  
-  Smart Insights  
-  Delete Last Entry  

---

##  How to Run the Project

### 1. Install Dependencies
`pip install matplotlib`

### 2. Run the Program
`python expense_tracker.py`
